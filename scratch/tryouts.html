CSS positions

css position include: fixed, static, sticky, relative, absolute

Static: This is according to the normal flow of the web page without any changes made

Relative: Positions the element relative to its original position

Fixed: An element is positioned relative to its viewport even when it is scrolled. The top, right, bottom, and left properties are used to position the element. A fixed element does not leave a gap in the page where it would normally have been located.

Sticky: An element is positioned sticky when it toggles between relative and fixed in relation to the viewport
